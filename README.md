## 📂 Topics Covered

### 01 — Array
> Fundamentals of arrays in C++

| File | Description |
|------|-------------|
| `ArrayAddition.cpp` | Adding elements to an array |
| `ArrayTraversal.cpp` | Traversing and printing array elements |

---

### 02 — Sorting
> Classic sorting algorithms with implementations

| File | Description |
|------|-------------|
| `BubbleSort.cpp` | Repeatedly swaps adjacent elements |
| `InsertionSort.cpp` | Builds sorted array one element at a time |
| `MergeSort.cpp` | Divide & conquer — O(n log n) |
| `QuickSort.cpp` | Partition-based — O(n log n) average |
| `SelectionSort.cpp` | Selects minimum and places in order |

---

### 03 — Searching
> Techniques to search for elements in data

| File | Description |
|------|-------------|
| `LinearSearch.cpp` | Sequential element-by-element search |
| `BinarySearch.cpp` | Efficient search on sorted arrays — O(log n) |
| `BinaryBubble.cpp` | Binary search combined with bubble sort |

---

### 04 — Linked List
> Dynamic linear data structure using nodes and pointers

| File | Description |
|------|-------------|
| `1-SinglyList.cpp` | One-directional node chain |
| `2-DoublyList.cpp` | Bidirectional node traversal |
| `3-CircularList.cpp` | Last node links back to head |

---

### 05 — Stack
> LIFO (Last In, First Out) data structure

| File | Description |
|------|-------------|
| `StackUsingArray.cpp` | Stack implemented with arrays |
| `StackUsingLinkedList.cpp` | Stack implemented with linked list |

---

### 06 — Queue
> FIFO (First In, First Out) data structure

| File | Description |
|------|-------------|
| `1-QueueUsingArray.cpp` | Queue with fixed-size array |
| `2-QueueUsingLinkedList.cpp` | Dynamic queue with linked list |
| `3-CircularQueue.cpp` | Efficient circular buffer queue |

---

### 07 — Tree
> Hierarchical non-linear data structure

| File | Description |

| `1-BinaryTree.cpp` | Basic binary tree operations |

| `4-LevelOrderTraversal.cpp` | BFS-based level-by-level traversal |

### 08 — Graphs
> Non-linear structure with vertices and edges

| File | Description |

### 09 — Recursion

## ⚙️ How to Run

```bash
# Clone the repository
git clone https://github.com/abdullah123-collab/DSA-Demo.git

# Navigate to any topic
cd "02 - Sorting"

# Compile with g++
g++ BubbleSort.cpp -o BubbleSort

# Run
./BubbleSort
```

> **Requirements:** Any C++ compiler — `g++`, `clang++`, or MSVC

# 🔍 Project — Mini Search Engine

> This folder is the origin of the **Mini Search Engine** project, developed as part of the DSA-Demo repository using **C++** and core Data Structures & Algorithms concepts.

---

## 📌 About the Project

The **Mini Search Engine** is a console-based C++ application that simulates how a real search engine works — loading documents, indexing words, and retrieving the most relevant results based on a user query.

It was built entirely using DSA concepts covered in this repository, making it a practical application of everything learned.

## 🧠 DSA Concepts Applied

| Concept | Usage in Project |
|--------|-----------------|
| **HashMap** | Word-to-document index mapping |
| **Linked List / Vectors** | Storing document lists per keyword |
| **Sorting** | Ranking results by relevance/frequency |
| **Searching** | Querying the index for keywords |
| **String Processing** | Tokenizing and parsing input documents |
| **Recursion** | Traversal and helper utilities |

---

## 💡 Features

- 📄 Load and index multiple text documents
- 🔎 Search any keyword and retrieve matching documents
- 📊 Results ranked by frequency of the keyword
- ⚡ Fast lookup using HashMap-based inverted index
- 🖥️ Clean console interface

## ⚙️ How to Run

> Full build instructions are available in the main repo linked above. Quick start:

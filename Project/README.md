## Project 1 — Mini Search Engine

The **Mini Search Engine** is a console-based C++ application that simulates how a real search engine works — loading documents, indexing words, and retrieving the most relevant results based on a user query. 

It was built entirely using DSA concepts covered in this repository, making it a practical application of everything learned.
